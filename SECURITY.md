# Security Policy

## Supported Versions

Currently, only the latest version of ArachnidForms is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1.0 | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within ArachnidForms, please do not disclose it publicly.
Instead, please send an email to the maintainers or open a private security advisory on GitHub.

We take security issues very seriously and will aim to respond as quickly as possible. When reporting a vulnerability, please include:
- A detailed description of the vulnerability.
- Steps to reproduce the issue.
- Any potential impact it might have on users.

Thank you for helping keep ArachnidForms secure!
